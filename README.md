# Cocodrilo
Cocodrilo es un lenguaje basado en FASM (assembler) que busca pooder ser muy eficiente y esta hecho para que niños de todo el mundo aprendan a programar en su lengua madre, la idea es ayudar a preservar Idiomas de cualquier país, credo o tradición vinculándolos a la tecnología sin obligar a nadie a cambiar su lengua para aprender.  es muy eficiente
